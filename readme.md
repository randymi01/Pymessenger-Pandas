# pymessage:
Simple python commandline texting

<br>

<b>Commandline Usage</b>
```
pymessage phonenumber message subject(opt) carrier(opt)
```
<br>
<b>Additional Usage Instructions:</b>
<br>
(optional) Add your own login details to config.json
